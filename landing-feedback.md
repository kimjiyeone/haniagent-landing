# 하니에이전트 랜딩페이지 전문가 피드백

## 현재 상태 분석 (리뉴얼 후)

### 잘 된 점
1. **히어로 구조**: tiro 스타일 2줄 슬로건 + 인터랙티브 데모 애니메이션이 제품 가치를 즉시 전달
2. **신뢰 요소**: 실제 한의원 로고 마퀴 2줄이 사회적 증거(Social Proof) 역할
3. **기능 테마별 분류**: 진료 자동화 / 환자 관리 / 경영 인사이트로 나눠 정보 과부하 방지
4. **보안 블랙 디자인**: 의료 데이터 보안의 심각성을 시각적으로 강조
5. **가격 투명성**: Basic(무료체험 2주) → Pro(가격문의) 구조가 진입 장벽을 낮춤

### 개선 제안
1. **앱 다운로드 URL**: 현재 haniagent.kr/auth/login으로 연결 — 실제 Windows 앱 다운로드 링크로 교체 필요
2. **리뷰 실명화**: 현재 "김OO 원장" 형태 — 가능하면 실명 + 사진으로 교체하면 신뢰도 상승
3. **데모 영상/GIF**: 인터랙티브 애니메이션 외에 실제 제품 화면 스크린샷이나 짧은 영상 추가 권장
4. **CTA 일관성**: "앱 다운로드" vs "무료 체험 시작" vs "1:1 맞춤 상담" — 주요 CTA를 하나로 통일 권장
5. **모바일 최적화**: 히어로 데모 애니메이션이 모바일에서 잘 보이는지 실기기 테스트 필요
6. **로딩 성능**: 로고 이미지 7개 + 목업 이미지 6개 → lazy loading 적용 확인 필요
7. **한의원 마퀴 2줄 중 하단**: 로고 없는 한의원이 많아 텍스트만 표시됨 — 실제 고객 한의원으로 교체하거나 제거 고려
8. **ComparisonSection**: Before & After 테이블이 기능 소개와 일부 중복 — 통합 또는 차별화 필요
9. **FAQ 섹션**: 현재 내용이 일반적 — "실제로 재진율이 올라가나요?", "기존 EMR과 호환되나요?" 등 구체적 질문 추가
10. **SEO**: meta description, OG tags, structured data 추가 권장
