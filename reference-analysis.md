# Reference Analysis

## tiro.ooo
- 히어로: 슬로건 + 2개 CTA 버튼 + 아래에 제품 스크린샷/영상 (실제 제품 화면이 자동 재생됨)
- Trusted by: 2줄 로고 그리드 (McKinsey, Bain, Airbnb, Sendbird, Moloco, QANDA, WoowaBros, Samsung, NAVER, Hyundai Card, Hyundai, Cheil, Dunamu, Smilegate, 서울대학교, KAIST)
- 로고는 흑백 처리, 2줄 정적 그리드 (마퀴 아님)
- 히어로 영상: 제품 UI가 실제로 작동하는 모습을 보여줌 (녹음 → 실시간 전사 → 요약 생성)

## mediark.app (심토미)
- 가격: Basic(무료, 가격 문의) + Premium(추천, 가격 문의)
- 가격 카드: 2열 그리드, Basic은 outline, Premium은 파란색 강조
- 기능 리스트: 체크마크 + 텍스트
- CTA: "무료로 시작하기" / "시작하기"

## 핵심 참고 포인트
- 히어로에 제품 작동 애니메이션을 크게 배치
- 로고 2줄 마퀴 (반대 방향)
- CTA: "무료로 시작하기" + "1:1 무료 컨설팅"
- 가격: 가격 문의 방식 (Basic + Premium)
