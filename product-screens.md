# HaniAgent 실제 제품 화면 분석

## 메인 화면 (원장 뷰)
- 상단: 하니 로고(H:) + 날짜(오늘 4월 2일 목요일) + 사용자명(김지연) + 원장/실장 전환 토글
- 대시보드 카드: 오늘 환자 6명 접수 / 미확정 차트 2건 확정 필요 / 인박스 처리율 0% (0/22건)
- 오늘 환자 리스트: 실시간 토글 + 저장 버튼 + 4/6건 확정
  - 강희범: Active · 1회차 · 두통, 이명 · 17:07 · 체크(확정완료)
  - 김순녀: Lite · 2회차 · 좌측 허리/둔부 통증 · 14:51 · 체크(확정완료)
- 완료환자 4건 (접기)
- 타임라인 21건
- 디자인: 깔끔한 화이트 배경, 민트/틸 브랜드 컬러, 카드 기반 레이아웃

## 실장 뷰
- 상단: 대시보드 동일 (오늘 환자 6, 미확정 차트 2, 인박스 처리율 0%)
- 퀵 체크인: 검색창 + [신규] 버튼 (민트색)
- 탭: 인박스 / EMR / 예약 현황
- 인박스 태스크 카드:
  - 김 · T1 현장안내 (노란 배경) — 메모: 리핏메모 — 카톡/안내불필요/X 버튼
  - 윤상필 · T1 현장안내 (노란 배경) — 메모: 통합보드 메모
  - 신태현 · T2 D-1 (파란 배경) — "내일" 표시 — 카톡 메시지 생성 중... — 메모: 리핏메모
- 완료 2건
- 타임라인 21건
- 디자인: T1=노란색 라벨, T2=파란색 라벨, 카톡 아이콘(노란 말풍선), X(삭제) 버튼

## EMR 탭 (차트 복붙 대기열)
- 윤상필 · 차트 라벨(노란) · 16:23 · 다한/손발·겨드랑이 땀 — 복사 아이콘 + > 버튼 — 메모: 통합보드 메모
- 김 · 16:59 · 리핏메모 — > 버튼
- 신태 · 리핏 라벨(노란) · 17:17 · 리핏메모 — > 버튼
- 박지은 · 리핏 라벨(노란) · 22:34 · 리핏 메모 — > 버튼
- 디자인: 차트=노란 라벨, 리핏=노란 라벨, 복사 아이콘, > 화살표로 상세 진입

## 통합보드 (환자 상세 - 원장 뷰)
- 상단: ← 목록 / 강희범 · 생성중(민트 라벨) · Active(노란 라벨) · 1회차 · 17:07
- 실장 전달 메모: textarea "실장에게 전달할 메모"
- 권장 방문일: [1일] [3일] [1주] [2주] [1달] [미정] [직접] 버튼 그룹
- 날짜 캐러셀: [오늘] 탭 선택됨
- SOAP 영역: "AI가 SOAP를 생성 중입니다..." 로딩 상태
- 디자인: 깔끔한 카드 레이아웃, 민트색 "생성중" 라벨, 노란색 "Active" 라벨
